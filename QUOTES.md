# Quote Catalog

This Markdown file is the source of truth for the `quotes` fortune database. Quotes are grouped by theme for readability. Each quote is a blockquote whose final line begins with an em dash for attribution. To regenerate the `quotes` file, flatten each blockquote (keeping any deliberate line breaks) and separate entries with `%`.
